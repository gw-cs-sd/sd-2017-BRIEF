This package contains the FZI icl_core package (independed core library) wrapped in a catkin package. This base library is used by other packages as a foundation.
It was developed at the FZI Research Center for Information Technology in Karlsruhe, Germany.

The build system "icmaker" is distributed under the BSD License.
The basis libraries "icl_core" is distributed under the LGPLv3 license.

Also included is the TinyXML Library (within the dependent package fzi_icl_can) which was developed by
Lee Thomason (http://www.grinninglizard.com)
and is distributed under the zlib/libpng license.

Find the license texts in this folder.
